# Bank Customer Analysis

## Project Overview
This project analyzes bank customer data using Python.  
It includes data exploration, visualization, and cleaning.  

## Technologies Used
- Python 3  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## Project Files
- `src/01_data_exploration.py` → Explore the dataset  
- `src/02_data_visualization.py` → Visualize data with charts  
- `src/03_data_cleaning.py` → Clean and encode data  
- `requirements.txt` → Python libraries  

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/Marzi-kh/bank_customer_analysis.git
2. install dependencies:
pip install -r requirements.txt
3. Run the script in order:
   1. `01_data_exploration.py`  
   2. `02_data_visualization.py`  
   3. `03_data_cleaning.py`
